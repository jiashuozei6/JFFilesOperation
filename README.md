# JFFilesOperation
iOS文件操作、文件夹操作等

## Example(示例)

运行案例工程，使用Git克隆响应地址，需先在Example文件夹运行`pod install`命令

## Requirements

## Installation(安装)

JFFilesOperation 可以通过 [CocoaPods](https://cocoapods.org)安装。在你的Podfile文件内，添加如下代码：

```ruby
source "https://github.com/jiashuozei6/PrivateSpecs.git"

pod 'JFFilesOperation'
```
